**Want JS to be easier? Use DynaScript!**

DynaScript is a JS library designed to simplify Graphics, Game Dev, DOM, and events. With DynaScript, you can make more interactive or **Dynamic** web apps.

DynaScript id simalar to jQuery and React, but there is some differences

jQuery Differences:

1. jQuery only has an alias for Query Selectors, DynaScript has one for all selectors

2. jQuery Query Selector is $(itemSelect); , while DynaScript's is $q(itemSelect);

React Differences:

1. React uses JSX or JavaScript XML, while DynaScript uses regular JS.

2. React uses createRoot(document.getElementById('root')).render(htmlContent), where it puts content generated by JavaScript in an HTML element, while DynaScript uses
   render(renderElementName,htmlContent), where it creates a new element, that always atarts with r-, and that is placed within an html file

ex. 

DyS/JS: render('abc', '< h1 > abc < /h1 >');

HTML: 
< r-abc >< /r-abc >
